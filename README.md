# Mandarin-Tool
**Advanced Multi-tool for Screen-share Analysis & System Diagnostics**

MandarinTool is a Java-based utility designed to streamline screen-share investigation.

📖 **[Check out the official Wiki](https://github.com/Mehmetyll/Mandarin-Tool/wiki) for guides and troubleshooting!**

## Core Features

### Analysis Modules
*   **Alt Checker:** Deep-scans account configurations for Lunar, Feather, SKlauncher, and Vanilla clients.
*   **Mod Analyze:** Scans and analyzes installed game/application mods.
*   **Class Analyze:** Advanced class search capabilities on analyzed mods.
*   **Jar Analyzer:** Forensic JAR inspector featuring integrated CFR decompilation and hex preview for deep archive analysis.
*   **USN Journal Analyze:** Tracks file and directory modifications via the NTFS USN Journal with advanced filtering.
*   **Prefetch Analyze:** Built-in parser for Windows Prefetch (.pf) files to reconstruct execution history.
*   **USB History:** Detailed history of USB device connections using event log (Partition/Diagnostic) and registry analysis.
*   **Regedit Analyze:** Scans critical Windows Registry keys and entries.
*   **EventLog Analyze:** Filters and displays specific Windows Event Logs.
*   **Recycle Bin Analyze:** Inspection of deleted files with detailed metadata recovery.
*   **JVM Analyze:** Detects and analyzes Java Virtual Machine executions.
*   **Services Analyze:** Enumerates and inspects Windows services, their statuses, and executable paths.
*   **Recent Analyze:** Lists recently accessed files and activities.
*   **CLI Analyze:** Analyzes command-line operations and execution history.
*   **PowerShell Analyze:** Checks local PowerShell command history.
*   **Site Bypass Analyze:** Analyzes bypass techniques and related web activity.
*   **Crash Dumps Analyze:** Inspects and analyzes crashed programs and dump files.

### Extra Features
*   **Scripts:** Run custom scripts directly via PowerShell integration.
*   **Tool Downloader:** Quickly download and manage other essential utilities.
*   **Language Support:** Supported Languages : 🇺🇸, 🇹🇷, 🇪🇸, 🇵🇱, 🇷🇺, 🇦🇿 .

## Getting Started
### Prerequisites
- Java Development Kit (JDK) 21 or higher.

###  Run [With CMD(administrator required for reach all features)]
```bash
java -jar MandarinTool.jar
```

## Or Run With [Mandarin-Launcher](https://github.com/Mehmetyll/Mandarin-Launcher) (fast and simple)

Run this command in CMD (as administrator):

```cmd
powershell -Command "Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass; Invoke-Expression (Invoke-RestMethod 'https://raw.githubusercontent.com/Mehmetyll/Mandarin-Launcher/main/Mandarin-Launcher.ps1')"
```
