# 🧰 Tool Downloader


Download forensic tools from various creators, all from one place.

---

## How to Use

1. Set your download path (defaults to your Downloads folder)
2. Expand a tool group to see available tools
3. Click **Download** next to a tool, or check multiple and click **Download Selected**
4. Click **Download All** to get everything

Tools are saved in a `Mandarin-DownloadedTools` subfolder, organized by group. ZIP files are auto-extracted.

Use 🔍 search to find specific tools. Click **Open Folder** to open the download directory.

---

## Available Tool Groups

| Group | Tools |
|-------|-------|
| **NirSoft** | Browsing History, Last Activity, Prefetch View, USB Log, Shell Bags, Jump List, and more |
| **NotRequiem & Nay-cat** | DoomsDeBunk, Jarabel, InjGen |
| **Spokwn** | Prefetch Parser, Journal Trace, BAM Parser, Process Parser, espouken, and more |
| **Orbdiff** | PrefetchView++, JournalParser, BAMReveal, USBDetector, Fileless, SSTool, and more |
| **Red Lotus** | Alt Checker, Mod Analyzer, Task Sentinel, Tool Downloader |
| **Echo** | Echo, BAM Deleted Keys, USB Device Log, User Assist, Journal |
| **Zimmerman** | PECmd, MFTECmd, Registry Explorer, AmcacheParser, TimelineExplorer, and more |
| **Other** | Everything, DIE, System Informer, Hayabusa, Luyten, FTK Imager, and more |

> ⚠️ Zimmerman tools require .NET 9 Runtime — download it from the last entry in that group.
> Echo tools open in the browser instead of downloading directly.
