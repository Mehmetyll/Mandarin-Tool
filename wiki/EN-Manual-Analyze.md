# 📂 Manual Analyze Tab


Contains four sub-tabs:

```
Mod Analyze · Class Analyze · JAR Analyzer · Zimmerman Assistant
```

---

## Mod Analyze

Scans folders for JAR/ZIP files and shows info about each one.

**How to use:**
1. Set a folder path (type it, click Browse, or pick from Templates)
2. Check options if needed — Recursive, Include ZIP, Extension Changed
3. Click **Analyze**

**Templates** give you quick presets for common mod folders (Lunar, Feather, Vanilla, etc.). You can select multiple.

**Results show:** File name, path, hidden status, size, extension changed, obfuscation level.

**Right-click a row** to open folder, copy name/path, or send it to JAR Analyzer.

**Search** uses `|` for OR: `vape | wurst | meteor`

---

## Class Analyze

Searches inside scanned mods for specific class names.

**How to use:**
1. Run Mod Analyze first
2. Type class names or use Templates (pre-built cheat signatures)
3. Click **Search**

Use `|` for multiple terms. Check **Exact Match** for exact name matching.

---

## JAR Analyzer

Deep inspection of a single JAR file. See [JAR Analyzer](EN-JAR-Analyzer) page.

---

## Zimmerman Assistant

Helper for running Eric Zimmerman's forensic tools. Download them first from the Download tab.
